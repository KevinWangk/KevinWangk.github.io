Mac M1 的VSCode C++ 无法输入的解决方法

发表于 2022-10-21

如何解决Mac M1 的VSCode C++ 无法输入

针对这个问题我做了很多改变，我并没有测试到底是哪一个方案是有效的，但是把这一些全部设置完是一定可以的。

参考链接：

https://zhuanlan.zhihu.com/p/353422349

https://blog.csdn.net/changyana/article/details/122699836

https://www.cnblogs.com/2018shawn/p/13580555.html

下面给出我做的详细改变：

c_cpp_properties.json

launch.json

tasks.json

以上是三个配置文件，接下来进行下一步：runInTerminal

首先是安装插件：Run in Terminal

接着找到左下角的设置：搜索runIn，会出现两行

找到两行：Run Code configuration 和 Run In Terminal Configuration

里面的选项全部勾选就可以了

# VSCode
C语言-sprintf-fscanf-EOF
